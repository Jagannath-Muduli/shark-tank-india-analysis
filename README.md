## Overview

This project analyzes Shark Tank India data to uncover insights about startup success rates, investment trends, and investor behavior.
Data was cleaned using SQL and visualized through an interactive Power BI dashboard.

## Tools & Skills

SQL – Data cleaning & transformation

Power BI – Dashboard & visualization

Excel / CSV – Dataset management

## Data Cleaning (SQL)

Steps performed:

Removed duplicates and missing values

Standardized data types (numeric, text)

Created calculated columns like success status and investment ratio

## Dashboard Highlights

Deal Status: 59.18% Success, 40.82% Failure

Top Sectors: Food, Lifestyle, Health Tech

Top Investors: Aman Gupta (₹799.93L), Peeyush Bansal (₹784.03L)

Equity Trends: Peeyush took the highest average equity (34.35%)

Gender Split: 139 Male, 59 Female participants

## Key Insights

Experienced investors like Aman and Peeyush dominate total funding.

Food & Lifestyle startups received the most investments.

Full-time founders were most successful in securing deals.
